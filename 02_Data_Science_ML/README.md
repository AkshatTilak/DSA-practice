# 02. Data Science & Machine Learning (ML)

This module focuses on theoretical and practical aspects of Machine Learning Engineering, Deep Learning architectures, and evaluation metrics used to measure model quality.

## 📂 Subdirectories
- `classical_ml/`: Linear and logistic regression, decision trees, random forests, boosting, bias-variance tradeoffs.
- `architectures/`: Structural details of deep learning models like Transformers (Attention mechanisms), Convolutional Neural Networks (CNNs), and Graph Neural Networks (GNNs).
- `systems_evaluation/`: Operational and statistical metrics (Precision, Recall, F1, ROC-AUC, MAPE, NDCG, log loss).

## ⚙️ Module Structure Guidelines
For coding-based components or mathematical functions:
1. `README.md`: Formal definitions, visual workflows, formulas, and production context (e.g., fraud pipelines).
2. `qXX_[name].py`: Executable functions representing the logic (e.g. customized metric solvers).
3. `qXX_test.py`: Pytest assertions to confirm calculations match theoretical values.
