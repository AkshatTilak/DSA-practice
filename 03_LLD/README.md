# 03. Low-Level Design (LLD)

This module compiles low-level design structures, object-oriented programming principles (SOLID), design patterns, and case studies representing common interview questions.

## 📂 Subdirectories
- `design_patterns/`: Implementations of Creational, Structural, and Behavioral patterns (Singleton, Strategy, Factory, Observer).
- `case_studies/`: Object models for complete systems (e.g. Warehouse Inventory Management, Parking Lot, Movie Ticket Booking).

## ⚙️ Module Structure Guidelines
1. `README.md`: Structural description, UML relationship representation (via ASCII/text), and trade-offs of patterns.
2. `qXX_[name].py`: Executable code modeling classes and interfaces.
3. `qXX_test.py`: Code validating thread-safety, object polymorphism, or behavioural flow.
